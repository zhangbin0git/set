设置缩进符为4个空格
===========================
  File -> Default Settings -> Code Style
  -> General -> 不勾选“Use tab character”
  -> Python -> 不勾选“Use tab character”
  -> 其他的语言代码同理设置

显示“行号”与“空白字符”
===================================
  File -> Settings  -> Editor ->General -> Appearance
  -> 勾选“Show line numbers”、“Show whitespaces”、“Show method separators”

设置IDE皮肤主题
================================
  File -> Settings -> Appearance -> Theme ->

设置编辑器“颜色与字体”主题
===================================
  File -> Settings -> Editor -> Colors & Fonts -> Scheme name -> 选择“Darcula”
  说明：先选择“Darcula”，再“Save As”一份，命名为“qianyunlai.com”，因为默认的主题是“只读的”，一些字体大小颜色什么的都不能修改，拷贝一份后方可修改！

修改字体大小
======================
  File -> Settings  -> Editor -> Colors & Fonts -> Font -> Size -> 设置为“14”

去掉默认折叠
============================
  File -> Settings  -> Editor -> Code Folding -> Collapse by default -> 全部去掉勾选 

“代码自动完成”时间延时设置
============================
  File -> Settings  -> Editor -> Code Completion 
  -> Auto code completion in (ms):0
  -> Autopopup in (ms):500

修改IDE快捷键方案
==============================
  File -> Settings  -> Keymap
  系统自带了好几种快捷键方案，下拉框中有如“defaul”,“Visual Studio”,在查找Bug时非常有用,“NetBeans 6.5”,“Default for GNOME”等等可选项，

设置单行最大字数
==================
File→Settings→Code Style→Right margin (columns) 的值为80，大功告成。
